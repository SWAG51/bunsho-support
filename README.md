# 文章生成サポート

Gmailの「Geminiで下書き」のように、伝えたい要点を入れるとビジネス向けの文章（LINE返信・お客様対応など）を作ってくれる、**デスクトップ常駐の小窓アプリ**です。頭脳は Google の Gemini（無料枠）。

## ダウンロード（インストール）

最新版は **[Releases ページ](../../releases/latest)** から：

| OS | ファイル |
|----|----------|
| **Windows** | `BunshoSupport-*-win-x64.exe` |
| **Mac (Apple Silicon / M1以降)** | `BunshoSupport-*-mac-arm64.zip` |
| **Mac (Intel)** | `BunshoSupport-*-mac-x64.zip` |

### Windows の開き方
ダブルクリック →「次へ」。「WindowsによってPCが保護されました」(SmartScreen) が出たら
**「詳細情報」→「実行」**。

### Mac の開き方（初回だけひと手間）
このアプリは Apple の有料の公証（notarize）を受けていないため、初回のみ手動で許可します。

1. `.zip` をダウンロードしてダブルクリックで解凍（`文章生成サポート.app` ができる）
2. その `.app` を **「アプリケーション」フォルダにドラッグ**
3. アプリを一度ダブルクリック → 「開けません」と出たら **キャンセル**
4. **「システム設定」→「プライバシーとセキュリティ」** を開く
5. 下のほうの「"文章生成サポート" は…ブロックされました」の右の
   **「このまま開く」** を押す → パスワード/指紋で許可
6. 次回からは普通にダブルクリックで開けます

> 古い macOS では、アプリを **右クリック →「開く」→「開く」** でも開けます。
> いずれも自作アプリのための警告で、危険なものではありません。

## 最初の1回だけ：Gemini APIキーの設定

1. アプリ右上の **⚙** →「無料APIキーの取得ページを開く」
2. Google でログイン →「APIキーを作成」→ コピー
3. アプリの欄に貼り付け →「接続テスト」→ OKなら「保存」

APIキーは各PC内だけに保存され、外部・このリポジトリには一切含まれません。

## 使い方

1. 上のタブで用途を選ぶ（LINE返信／お客様対応 など）
2. 「相手から届いた文」（あれば）と「伝えたい要点」を入力
3. 「文章を作る」→「コピー」して LINE やメールに貼る（「作り直す」で別案）

PC起動時に自動でトレイ常駐します（トレイ右クリックで切替）。

## 開発・ビルド

```bash
npm install        # 準備
npm start          # 起動（開発）
npm run dist:win   # Windowsインストーラを作る（release/ に出力）
```

Mac版はタグ（例 `v1.0.0`）を push すると GitHub Actions が自動でビルドし、Releases に添付します。
