==================================================
 文章生成サポート ― 配布（インストーラ）手順
==================================================

■ Windows版（このPCで作成済み）
 ファイル: release\BunshoSupport-1.0.0-win-x64.exe
 これを相手に渡す（USB・メール添付・クラウド共有など）。
 受け取った人は ダブルクリック →「次へ」で進めるとインストール完了。
 ・初回起動時に「WindowsによってPCが保護されました」(SmartScreen)が
   出ることがある →「詳細情報」→「実行」で進めればOK。
   ※ 署名証明書を入れていない未署名アプリのため。怪しいものではありません。
 ・インストールするとスタートメニュー/デスクトップにアイコンができ、
   次回PC起動時から自動でトレイに常駐します（トレイ右クリックで切替可）。

 再ビルドしたいとき（中身を直した後）:
   コマンドで  npm run dist:win   → release フォルダに新しい .exe ができる。

--------------------------------------------------

■ Mac版（mac実機が無いのでクラウドで自動ビルド）
 Windows上ではMacの.dmgは作れないため、GitHubのクラウド(Mac環境)で作ります。
 .github\workflows\build.yml が設定済みです。

 手順（最初の1回だけ）:
  1. このフォルダをGitHubのリポジトリにpushする
  2. GitHubの「Actions」タブ →「build-installers」→「Run workflow」を実行
  3. 数分待つと、WindowsとMacのインストーラが「Artifacts」に出来る
     - installer-mac   … .dmg / .zip（Mac用）
     - installer-windows … .exe（Windows用）
  4. ダウンロードして配布

 Mac版の注意:
  ・未署名のため、初回は「開発元が未確認」と出る →
    アプリを右クリック →「開く」→「開く」で起動できる（1回だけ）。

==================================================
 ※ APIキー（Gemini）は各PCで初回に1回だけ設定。
   インストーラには鍵は含まれません（各自で取得・保存）。
